# Migration Guides

> **⚠️ IMPORTANT: DO NOT SKIP VERSIONS**
>
> When updating your conference website, you **MUST** apply migration guides sequentially.
>
> **Example:**
> If you are on `v1.0.0` and want to update to `v1.2.0`:
>
> 1. First, apply the changes from `v1.0.0` → `v1.1.0`.
> 2. Then, apply the changes from `v1.1.0` → `v1.2.0`.
>
> **Do not skip intermediate steps**, as changes are often cumulative. Skipping a version may result in broken features, missing configuration, or styling issues.

## Available Guides

Please check the `guides/` subdirectory for specific version migration instructions.
