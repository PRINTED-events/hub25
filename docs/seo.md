# SEO Features & Configuration

This template comes with SEO capabilities powered by [`@nuxtjs/seo`](https://nuxtseo.com/).

## Features Overview

- **Automatic Sitemap**: Generated at `/sitemap.xml`.
- **Sitemap Freshness**: With `sitemap.zeroRuntime` enabled in `nuxt.config.ts`, `/sitemap.xml` updates on deployment, so Nuxt Studio content saves must trigger an automatic rebuild/deploy.
- **Robots.txt**: Pre-configured to allow legitimate crawlers but block AI training bots.
- **Dynamic OG Images**: Social media cards are automatically generated for every page (Talks, Speakers, Landing Page, etc.).
- **Schema.org**: Automated structured data.
- **Metadata**: Automatically managed Metadata.
